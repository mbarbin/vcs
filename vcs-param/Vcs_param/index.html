The initialization should be created very early in the commands body. The cwd shall not be changed subsequently.

Params

Some parameters may only be created under a certain context, in that case they're exposed wrapped under a resolvable type. Otherwise they can be exposed as command params directly.

If not otherwise specified, the parameters are resolved under the current context, and are required. Optional parameters ends with _optional.

type 'a t

A 'a Vcs_param.t is a command line parameter that produces a value of type 'a under the initialized context.

val resolve : 'a t -> context:Context.t -> 'a Base.Or_error.t

To be called in the body of the command, after initialization.
