Module Vcs.Refs

Capturing the information related to git refs.

In this file we represent the output of the git show-ref command. In particular we represent the location of all known local branches heads, remote branches heads and tags.

It is worth noting that the location of the remote branches is the state of the remote branches as they are currently known locally. To connect to the remote repository and retrieves information about the remote references, see Fetch_remote.
