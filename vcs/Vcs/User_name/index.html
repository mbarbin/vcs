Module Vcs.User_name

User name information as specified by the Git config user.name value.

Usually, this includes the full first name and last name, such as User_name.v "John Doe". This is not to be conflated with the User_handle, which are used as GitHub logins and other contexts.

val of_string : Base.string -> t Base.Or_error.t

of_string str returns Ok str if X.invariant str = true, and an error otherwise. This is meant to be used to validate untrusted entries.

val v : Base.string -> t

v str is a convenient wrapper for of_string str |> Or_error.ok_exn. This is typically handy for applying on trusted literals.
